Загрузка калькулятора…

Если это сообщение остаётся дольше нескольких секунд, попробуйте по очереди:

  1. Жёсткий reload: Ctrl+Shift+R (Windows/Linux) или Cmd+Shift+R (macOS) — браузер агрессивно кэширует ES-модули, обычный F5 может подгрузить старую версию.
  2. Открыть DevTools (F12) → вкладка Console. Ошибки загрузки модулей покажут точную причину.
  3. Если адрес начинается с file:// — ES-модули заблокированы политикой CORS. Запустите локальный HTTP-сервер:
    • Windows: двойной клик по start-server.bat
    • Linux / macOS: bash start-server.sh
    • Вручную: python -m http.server 8000 или npx http-server -p 8000, затем http://localhost:8000.

Подробнее — в файле HOW_TO_START.md.
