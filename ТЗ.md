## 1. Назначение

Расчёт инфраструктуры (5 стендов: DEV/IFT/PSI/PROD/LOAD) и стоимости (₽/мес, ₽/год) по опроснику. Универсальность (алгоритмы в коде, данные снаружи), market-grade UX, простота (2 мин на расчёт), надёжность (undo + graceful errors). [hirehi](https://hirehi.ru/blog/rabota-s-griaznymi-dannymi-kak-chistit-normalizovat-i-ne-poteriat-vazhnuiu-informatsiiu)


## 5. Алгоритмы

**Формулы**:
```
qty(ЭК,стенд) = eval(ЭК.qtyFormulas[стенд])
costMonthly = qty × price × tariffMul(S.phaseDurationMonths)
costFinal = costMonthly × (1+bufferTask) × (1+bufferProject) × (1+indexation)
totalStand = Σ costFinal(applicable ЭК)
totalAll = Σ totalStand
```
tariffMul: monthly=1, annual=1/12, oneTime=1/phase(≤0→12).

**DSL** (собственный парсер): Q.id, S.param, STAND, true/false, min/max/if/арифметика/логика. Edge: /0→0, NaN→{qty:0,error}. [habr](https://habr.com/ru/articles/1011524/)
