# Как считается бюджет

Этот документ описывает пользовательскую логику расчёта без внутренней
технической терминологии.

## 1. Что такое ЭК

ЭК — элемент конфигурации: процессоры, память, диски, трафик, объектное
хранилище, WAF, лицензии, работы по внедрению и другие статьи инфраструктуры.

У каждого ЭК есть:

- единица измерения: vCPU, ГБ, ТБ, штука, мероприятие, 1 млн токенов;
- цена за единицу;
- период тарифа: день, месяц, год или разовый платёж;
- формула количества для каждого стенда.

## 2. Откуда берётся количество

Количество ЭК считается из ответов пользователя и параметров расчёта.

Источники ответов:

- из Quick Start — пользователь выбрал быстрый профиль, приложение заполнило
  ответы автоматически;
- из опросника — пользователь ввёл или изменил ответ вручную;
- по умолчанию — пользователь не менял поле, взято стандартное значение.

Пример:

```text
Количество RAM = число серверов × RAM на сервер × коэффициент размера стенда
```

Если пользователь меняет один ответ, могут измениться сразу несколько ЭК.
Например, рост нагрузки влияет на CPU, RAM, диски, трафик и часть сервисов.

## 3. Как количество превращается в деньги

Для каждого ЭК и стенда считается базовая стоимость за месяц:

```text
количество × цена за единицу × месячный множитель тарифа
```

Месячный множитель тарифа:

- месячный тариф: `×1`;
- дневной тариф: `×количество дней в месяце`;
- годовой тариф: `×1/12`;
- разовый платёж: делится на длительность расчёта в месяцах.

После этого применяются риск-коэффициенты и НДС:

```text
итог за месяц = базовая стоимость × риски × НДС
```

Риски и НДС разделены. НДС — налог, а не риск, поэтому он включается отдельной
настройкой.

## 4. Почему важны единицы измерения

Единица количества и единица цены должны совпадать.

Примеры:

- если количество в ТБ, цена должна быть за ТБ;
- если количество в `1 млн токенов`, цена должна быть за 1 млн токенов;
- если количество в `1000 SMS`, приложение показывает это как `тыс. SMS`;
- трафик обычно считается за месяц, а SSD/HDD/S3 — как размер хранилища.

В Детализации блок `Проверка расчёта ЭК` показывает контроль единиц. Статус
`ОК` означает, что цепочка `qty × цена × период × риски × НДС` сходится с
итоговой стоимостью.

## 5. Где смотреть проверку

Вкладка `Детализация` показывает блок `Проверка расчёта ЭК`.

В нём выводятся top-10 самых дорогих ЭК текущего расчёта:

- вклад ЭК в бюджет за месяц и за год;
- расчёт по самому дорогому стенду;
- источники ответов: Quick Start, опросник, значение по умолчанию;
- проверка единиц измерения.

Для полного разбора конкретной строки нажмите `Почему столько?` рядом с ЭК.
Откроется окно с ответами, коэффициентами стенда и формулой количества.
